# Network Traffic Anomaly Detection

This project implements an unsupervised anomaly detection system for network traffic analysis using Isolation Forest. The model identifies unusual packet behavior without requiring labeled attack data, making it suitable for real-world cybersecurity monitoring.

## Features
- Network traffic preprocessing and cleaning
- Feature engineering from packet metadata
- Protocol, source, and destination encoding
- Isolation Forest-based anomaly detection
- Anomaly scoring and ranking
- Traffic visualization and analysis

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Applications
- Intrusion Detection
- Cybersecurity Monitoring
- Network Traffic Analysis
- Outlier Detection
